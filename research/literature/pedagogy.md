# Pedagogy — Literature Map

**Status:** Outline. Ready for population by reviewer.

## Core question
What actually works in language learning, and does SRE's experience
model align with evidence-based pedagogy?

## Current references
- [25] Krashen (1985) — The Input Hypothesis
- [26] Swain (2005) — The Output Hypothesis
- [27] Nation (2007) — The Four Strands

## Mapping SRE to pedagogy

| Pedagogical principle | SRE mechanism | Evidence base |
|---|---|---|
| Comprehensible input | Semantic density metric (0.7 ent/sent) controls complexity | Krashen (i+1) |
| Pushed output | Entity reuse ratio (0.6) forces repeated encounters | Swain |
| Spaced repetition | Experience list is sequential; no spaced repetition yet | Ebbinghaus |
| Contextual learning | Experiences embed entities in narratives | Nation (meaning-focused input) |
| Explicit learning | Vocabulary lists available per experience | Nation (language-focused learning) |
| Fluency development | Multiple experiences reuse same entities | Nation (fluency) |

## Gaps
- No spaced repetition algorithm. SRE orders experiences by level but
  does not schedule retrieval practice.
- No assessment or testing component. SRE delivers content but does
  not measure learning outcomes.
- No personalisation. All learners see the same sequence.
- No output enforcement. Experiences present text but do not require
  production.

## Research lineage note
> SRE's pedagogical model is informal and pragmatic: it embeds entities
> in narrative contexts and reuses them across experiences. This aligns
> with meaning-focused and usage-based approaches but has not been
> tested against alternatives. The pedagogical claims in the white
> paper should be read as design hypotheses, not empirical findings.
